# Task-Management-App

A personal Android project focused on UI implementation using Material 3, inspired by a task management app design originally found on Dribbble.
Rather than a pixel-perfect recreation, this app adapts and reinterprets the original design to better align with Material 3 guidelines, Android interaction patterns, and Jetpack Compose best practices.

The goal is to take a visually appealing concept and translate it into a functional, idiomatic Android UI, making design decisions where the original concept does not naturally fit the Material ecosystem.
